# Hani Ashour Portfolio — V2

Interactive, bilingual, dependency-free portfolio for GitHub Pages.

## Publish
1. Open the existing GitHub Pages repository.
2. Replace the old `index.html` and `assets` folder with this version.
3. Commit changes.
4. Wait 1–3 minutes and press `Ctrl + F5` on the live site.

## Features
- Famcare as a major career chapter and featured project
- Mouse-follow data network and glow
- Magnetic buttons and 3D project-card tilt
- Scroll reveals and animated metrics
- Interactive case-study drawer
- English/Arabic toggle
- Reduced-motion accessibility support

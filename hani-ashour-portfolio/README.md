# Hani Ashour Portfolio

## Free deployment with GitHub Pages

1. Sign in to GitHub.
2. Create a **public repository** named exactly: `YOUR-USERNAME.github.io`
3. Upload all files and folders from this package to the repository root.
4. Open **Settings → Pages**.
5. Choose **Deploy from a branch**.
6. Select `main` and `/ (root)`, then save.
7. Your free website will be available at `https://YOUR-USERNAME.github.io/`.

## After publishing

- Add the website URL to LinkedIn Contact info.
- Add the website as the first item in LinkedIn Featured.
- Submit the URL to Google Search Console.
- Once the exact URL is known, add a canonical link and sitemap.
